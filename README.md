# Block-Letters

Displays my initials on the screen in block letters and creates an ASCII art piece.<br/>



```#include <iostream>

int main() {

  std::cout << "DDDD   BBBB\n";
  std::cout << "D   D  B    B\n";
  std::cout << "D   D  B    B\n";
  std::cout << "D   D  BBBB\n";
  std::cout << "D   D  B    B\n";
  std::cout << "D   D  B    B\n";
  std::cout << "DDDD   BBBB\n";
}
```
